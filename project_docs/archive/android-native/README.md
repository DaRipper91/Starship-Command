# Star Command Native (Kotlin)

Native Android rewrite for Star Command using Kotlin + Jetpack Compose.

## Stack

- Kotlin 1.9
- Android Gradle Plugin 8.5
- Jetpack Compose (Material 3)
- DataStore Preferences
- Kotlinx Serialization

## Features in this module

- Build screen for core theme fields
- Preview screen for prompt simulation
- Export screen for `starship.toml` output
- Local persistence for current theme draft

## Build

```bash
./gradlew assembleDebug
```
